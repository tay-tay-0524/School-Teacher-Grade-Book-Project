# School-Teacher-Grade-Book-Project
This project is mainly script-only python coding. The purpose of this project is use python programming to building what a teacher's grade book might look like for a class.


This is project will include the following (outside python code):
- A list of student names with their grade for a specific subject next to them; separated by subject
- A list of assignments needed to be collected or graded
- Features for the teacher to see. This includes the following:
    - See who is attending class
    - What grade that SPECIFIC student has in that SPECIFIC subject
    - Change a student's grade  (doesn't matter up or down)
    - see student's ID number
